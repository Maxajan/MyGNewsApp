My GNews App
A modern Android news application built with Kotlin and Jetpack Compose, following the MVVM architecture pattern.  
It fetches live news from the GNews API, supports search, categories, and favorites, and stores saved articles locally using Room.

Features

-  **MVVM Architecture** — clean separation of UI, business logic, and data  
-  **Retrofit** — for smooth API communication  
-  **Room Database** — local storage for favorite articles  
-  **Hilt** — dependency injection for scalable code  
-  **ViewModel + StateFlow** — efficient state management  
-  **Jetpack Compose** — fully modern UI toolkit  
-  **Search functionality** — find news by keyword  
-  **Save favorites** — read your favorite articles anytime
